ead-finder
==========

ead-finder is a small Node utility that looks for EAD XML that has been 
indexed by Google.

Install
-------

1. sudo apt-get install nodejs redis-server
1. git checkout https://github.com/edsu/ead-finder.git
1. cd ead-finder
1. npm install
1. node crawl.js
1. redis-dump --json > dump.json
